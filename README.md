# Weather App 🌤️

A simple and clean weather application built using **HTML, CSS, and JavaScript**.

## 📌 About the Project
This Weather App allows users to enter a city name and get **real-time weather information** such as temperature and current weather condition.  
The app fetches live data from an external weather API and updates the UI dynamically.

## ✨ Features
- Search weather by city name
- Displays real-time temperature
- Shows current weather condition
- Handles invalid city names gracefully
- Clean and responsive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- WeatherAPI

## ⚙️ How It Works
1. User enters a city name
2. On button click, an API request is made
3. Weather data is fetched using `fetch` and `async/await`
4. The result is displayed on the screen

## 🔑 Note
This project uses a **free API key** for learning and demonstration purposes.  
For production applications, API keys should be handled securely on the backend.
